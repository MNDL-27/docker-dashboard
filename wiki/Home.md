## üéØ About Docker Dashboard

Docker Dashboard is a fully containerized web application for monitoring and managing Docker containers. It features:

- ‚úÖ **Real-time monitoring** - CPU, RAM, Network, Disk usage with live charts
- ‚úÖ **Bandwidth tracking** - Total data downloaded/uploaded by each container
- ‚úÖ **Container management** - Start, stop, restart with one click
- ‚úÖ **Log streaming** - Live container logs via WebSocket
- ‚úÖ **qBittorrent integration** - Accurate stats for VPN-bound containers
- ‚úÖ **Portainer support** - Multi-host Docker management
- ‚úÖ **Modern UI** - Glass-morphism design with TailwindCSS
- ‚úÖ **Zero dependencies** - Fully containerized, no Node.js required
